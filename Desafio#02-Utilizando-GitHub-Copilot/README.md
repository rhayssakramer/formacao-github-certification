### ▶️ Introdução
Neste desafio, vamos explorar o GitHub Copilot como assistente para resolver problemas em Python, incluindo manipulação de strings, operações matemáticas, verificação de números pares e ímpares, cálculo de médias e identificação de palíndromos.

### 🎯 Desafio de Projeto
O objetivo é testar suas habilidades em Python utilizando sugestões de IA para:
- Manipular strings e números
- Criar algoritmos simples e eficientes
- Aplicar boas práticas de programação

>**Atenção!** Não tem acesso ao Github Copilot?! Não tem problema! Que tal utilizar o ChatGPT como seu copiloto de estudos?

### 🛠️ Objetivos
O objetivo deste desafio é:
- Aprender a utilizar `GitHub Copilot` para sugerir soluções em Python.
- Desenvolver `algoritmos simples e eficientes`.
- Explorar conceitos de `manipulação de strings, números, condicionais e loops`.
- Entender como aplicar `IA como assistente de programação`.
- Treinar `processos de colaboração` e versionamento com Git e GitHub

#### 📌 Pré-requisitos
1. Ter **Python** instalado na máquina (recomendado Python 3.10+).
2. Editor de código: VS Code ou outro compatível.
3. Acesso ao **GitHub** e repositório criado para o desafio.
4. Git instalado e configurado localmente.
5. (Opcional) Acesso ao **GitHub Copilot** ou ChatGPT como copiloto.

#### 🧠 O que será feito?
No desafio, você irá:
1. Concatenar Dados
- Receber dois dados do usuário e uni-los em uma única string.
- Aprender: manipulação de strings, concatenação, entrada de dados.
2. Repetir Textos
- Solicitar uma string e um número inteiro, retornando a string repetida.
- Aprender: manipulação de strings, números inteiros, loops e entrada de dados.
3. Operações Matemáticas Simples
- Receber dois números e realizar operações básicas.
- Aprender: operadores matemáticos, entrada de dados, lógica de cálculo.
4. Verificar Números Pares e Ímpares
- Receber um número inteiro e determinar se é par ou ímpar.
- Aprender: condicionais, operador módulo (%), lógica de verificação.
5. Calcular Média de Notas
- Receber três notas e calcular a média.
- Aprender: variáveis, operadores aritméticos, entrada de dados.
6. Verificar Palíndromos
- Testar se uma palavra é palíndromo.
- Aprender: manipulação de strings, reversão de texto, comparação e lógica de palíndromos.

#### 🚀 Passo a Passo
1. Clone o repositório na sua máquina local.
2. Abra o projeto no VS Code ou outro editor compatível com Python.
3. Explore os arquivos de cada desafio.
4. Utilize o GitHub Copilot para gerar sugestões de código.
5. Teste as soluções executando os scripts no Python.
6. Compare os resultados com as expectativas descritas em O que será feito?
7. Faça alterações, commits e aprenda a iterar com o GitHub.

O que aprenderemos?
- Manipulação de Strings (string)
- Concatenação
- Entrada de dados

### ✏️ Repetindo Textos
Descrição: Agora vamos solicitar uma string e um número inteiro como entrada. Depois teremos que retornar a string repetida o número de vezes informado.

O que aprenderemos?
- Manipulação de Strings (string)
- Números Inteiros (int)
- Múltiplas repetições
- Entrada de dados
- Aproveitar as sugestões do Github Copilot

### 📐 Operações Matemáticas Simples
Descrição: Vamos solicitar como entrada dois números e depois vamos realizar uma operação simples entre eles.

O que aprenderemos?
- Operações Matemáticas Básicas
- Entrada de dados
- Utilização eficiente do Github Copilot

### 🧮 Verificando Números Pares e Ímpares
Descrição: Como entrada, receba um número inteiro e verifique se ele é par ou ímpar. Uma dica é: Utilize condicionais para realizar a verificação e, se possível, faça uso do Github Copilot(ou outra IA) para otimizar a estrutura do código.

O que aprenderemos?
- Utilização de condicionais em Python (if, else) para realizar verificações.
- Introdução ao conceito de operador de módulo (%) para verificar se um número é par ou ímpar.
- Exploração do uso de uma ferramenta de IA, como o Github Copilot, para otimizar a estrutura do código.


### 📚 Calculando Média de Notas
Descrição: Agora vamos calcular a média de três notas fornecidas na entrada do usuário. Uma dica é: Utilize operadores aritméticos para realizar o cálculo da média.

O que aprenderemos?
- Uso de variáveis para armazenar dados fornecidos pelo usuário.
- Aplicação de operadores aritméticos (+, /) para calcular a média de um conjunto de valores.
- Prática na solicitação e manipulação de entrada do usuário.

### 🔄 Verificando Palíndromos
Descrição: Vamos testar se uma palavra é um palíndromo?!
Uma dica é: Utilize conceitos de manipulação de strings para inverter a palavra e comparar com a original.

O que aprenderemos?
- Manipulação de strings em Python, especialmente invertendo uma string.
- Compreensão de como comparar a string original com sua versão invertida para determinar se é um palíndromo.
- Introdução ao conceito de palíndromos e sua aplicação em problemas de programação.
